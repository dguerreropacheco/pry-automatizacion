# Automatización APIS - SWAGGER PETSTORE 2.0 / ENDPOINT: USER

Se implementaron:

- Crear Usuario
- Obtener Usuario por Username
- Actualizar Usuario 
- Eliminar Usuario
- Adicional: Step de Login para simular la precondición en crear , actualizar y eliminar usuario.
